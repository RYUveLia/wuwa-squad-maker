# WuWa Matrix Squad Maker

명조: 워더링 웨이브의 핵심 도전 콘텐츠인 **종말 매트릭스(Tower of Adversity / Adversity Matrix)** 다중 파티 구성을 시각적으로 시뮬레이션하고 간편하게 공유할 수 있는 웹 기반 파티 빌더 도구입니다.

---

## ✨ 주요 기능 (Key Features)

1. **직관적인 드래그 앤 드롭 빌더 (Drag & Drop Builder) (PC 전용)**
   * 좌측 도감에서 원하는 공명자 카드를 끌어다 우측 파티 슬롯(딜러, 서브, 힐러)에 간편하게 배치할 수 있습니다.
   * 슬롯 간에 직접 끌어다 놓아 파티원의 **위치를 서로 교환(Swap)**할 수 있습니다.
   * 배치된 캐릭터를 클릭하거나, 슬롯 밖(도감 영역 등)으로 드래그 앤 드롭하면 배치가 **즉시 해제**됩니다.
   * 드래그 핸들러를 사용해 **파티 행(Row) 단위로 자유롭게 순서를 드래그 정렬**할 수 있습니다.

2. **모바일 탭 조작 최적화 & 연속 편성 바텀 시트 (Mobile Only)**
   * 모바일(1024px 미만)에서는 무하한 스크롤의 피로를 유발하던 드래그앤드롭 대신, **슬롯 터치 기반의 전용 캐릭터 선택 팝업창(바텀 시트)**이 자동으로 마운트됩니다.
   * 1인 배치 마다 창이 매번 닫히지 않고, 3명이 꽉 찰 때까지 팝업을 띄워두는 **연속 탭 편성**을 지원합니다.
   * 팝업 내 상단 **미니 3슬롯 실시간 프리뷰**를 통해 초상화를 가볍게 터치해 배치를 실시간 취소할 수 있으며, 1~2명만 구성하고 닫을 수 있는 **'편성 완료' 확인 버튼**을 탑재했습니다.

3. **종말 매트릭스 출전 제한 규칙 완벽 자동화**
   * **서포터 최대 2회 중복 출전 허용:** 설지, 복링, 벨리나, 파수인, 모니에 등의 서포터 공명자는 최대 2개 파티까지 중복 편성이 가능하며, 한도를 초과하면 FIFO(선입선출) 방식으로 기존 배치를 비우고 똑똑하게 재배치합니다. (유출 정보 토글 설정에 따라 유출 정보 대상 공명자의 중복 편성 규칙도 유기적으로 자동 전환됩니다.)
   * **시즌 특별 규칙 적용 (치사):** 3.5 시즌 임시 버프 조건인 **"치사 최대 2회 중복 출전"** 룰을 기본 탑재하여, 별도 조작 없이 치사도 최대 2회 배치가 가능하도록 자동 제어합니다.
   * **방랑자 4속성 배타성 잠금 (Exclusion Lock):** 회절, 인멸, 기류, 전도 방랑자 중 하나라도 파티에 배치되면 나머지 3개 속성의 방랑자 카드는 도감에서 자동으로 딤드(비활성화) 잠금 처리되어 중복 출전이 방지됩니다.

4. **보유 공명자 필터링 시스템 (Owned Filter)**
   * 내가 보유 중인 공명자들만 체크(✓)하여 관리할 수 있는 **보유 공명자 설정 모달**을 지원합니다.
   * 설정한 데이터는 브라우저 `localStorage`에 영구 기록되어 새로고침 후에도 유지됩니다.
   * "보유한 공명자만 보기" 필터 스위치를 켜면, 미보유 캐릭터들은 리스트에서 제외되어 한층 기동력 있는 파티 빌딩이 가능합니다.

5. **유출 정보 ON/OFF 슬라이딩 토글 스위치 (Leak Filter Toggle)**
   * 미출시 유출 공명자의 노출 상태 및 제한 규칙을 손쉽게 제어할 수 있는 슬라이딩 토글 스위치를 지원합니다.
   * 토글 ON 시 유출 설정 대상 공명자는 최대 2회 중복 출전이 허용되며, OFF 시에는 1회 출전으로 제한이 자동 연동됩니다.
   * 설정한 토글 데이터 또한 브라우저 `localStorage`에 보존되어 새로고침 후에도 안정적으로 유지됩니다.

6. **만석 시 자동 파티 증설 시스템**
   * 모든 파티의 빈 슬롯이 가득 찬 상태에서 도감의 공명자 카드를 클릭했을 때, "편성할 빈 슬롯이 없습니다. 새로운 파티를 추가하고 배치하시겠습니까?" 라는 유려한 컨펌 모달이 출력됩니다.
   * 확인 버튼을 누르면 자동으로 파티(Row)가 한 줄 신규 생성(증설)되면서 해당 캐릭터가 1번 슬롯(딜러)에 즉시 배치되어 끊김 없는 조작을 유도합니다.

7. **실수 방지용 확인 팝업 및 키보드 단축키 매핑 (ConfirmModal & Keyboard Shortcuts)**
   * 파티원 해제, 개별 파티 줄 삭제, 툴바 전체 초기화 버튼, 그리고 만석 시 파티 추가 시 소중하게 짜놓은 스쿼드 배치가 우발적인 터치로 해제되는 비극을 예방하기 위해 아름다운 글라스모피즘 스타일의 **확인 컨펌 팝업창**을 경유하도록 설계했습니다.
   * 팝업 내부의 단어 쪼개짐 방지(`break-keep`) 및 모바일 해상도 버튼 찌그러짐 차단(`whitespace-nowrap flex-shrink-0`)을 적용해 비주얼 완성도를 극대화했습니다.
   * 사용성 극대화를 위해 팝업창이 활성화된 상태에서 **`Enter` 키**를 누르면 즉시 승인(Confirm) 처리되고, **`Escape` 키**를 누르면 취소(Cancel) 처리되는 키보드 단축키 매핑을 완벽 지원합니다.

8. **Cloudflare Turnstile 봇 보호 방화벽 (Bot Protection)**
   * Vercel 무료 도메인 환경을 겨냥한 악성 스크랩 봇이나 디바이스 폭격을 원천 차단하는 지능형 **Turnstile 비가시형 캡차 게이트웨이**를 소스 코드 단에 완벽 탑재했습니다.
   * 광고 차단기(AdBlock 등)로 인해 캡차 API가 강제 차단당하는 억울한 진짜 사용자를 위해, 하단에 비상 탈출용 **[보안 인증 건너뛰기]** Fail-safe 링크를 동봉하여 조화로운 보안을 성취했습니다.
   * 환경 변수(`VITE_TURNSTILE_SITE_KEY`) 및 로컬 런타임 하드코딩 교체를 모두 지원합니다.

9. **고화질 파티 캡처 이미지 저장 & Base64 코드 내보내기/불러오기**
   * 구성한 조합 배치도를 고해상도 PNG 이미지로 간편하게 내려받을 수 있습니다.
   * 영문 캐릭터 ID 기반의 Base64 인코딩 스트링 코드 공유를 지원하여, 간편하게 파티를 전파하고 역으로 복원해 올 수 있습니다. (레거시 한글/영문 텍스트 코드 또한 완벽 하위호환 지원)

## 🛠️ 기술 스택 (Tech Stack)

* **프레임워크:** React 19, Vite 8, TypeScript 6
* **스타일링:** Tailwind CSS 4
* **드래그 앤 드롭:** `@dnd-kit/core`, `@dnd-kit/sortable`, `@dnd-kit/utilities`
* **스크린샷 캡처:** `html-to-image`

## 🏛️ 아키텍처 및 코드 품질 (Architecture)

관심사의 완벽한 분리(SoC)와 컴포넌트 독립성을 최우선으로 리팩토링된 견고한 구조를 갖추고 있습니다:
* **UI View Layer ([src/App.tsx](src/App.tsx)):** 복잡한 하위 컴포넌트 구현이 분리되어, 메인 그리드 레이아웃과 렌더링 스크린 마운트에 집중하는 정갈한 메인 뷰 파일입니다.
* **Logic Controller ([src/hooks/useSquadState.ts](src/hooks/useSquadState.ts)):** 스쿼드 상태값 제어, 드래그앤드롭 서브 핸들러 세분화, FIFO 알고리즘 계산, 클립보드 복사 등 모든 비즈니스 도메인 규칙을 캡슐화한 커스텀 훅입니다.
* **독립 컴포넌트 모듈:** `ImportModal`, `SortableSquadRow`, `Toast`, `DraggableCharacterCard`, `DroppableSquadSlot`, `DroppableCharacterPool` 컴포넌트들이 개별 파일로 완벽하게 독립되어 가독성과 재사용성이 크게 증대되었습니다.
* **Component-Style Colocation:** 전역 스타일 시트 의존도를 줄이기 위해, 각 독립 컴포넌트 하단부에 Tailwind CSS 전용 스타일 상수를 자가 포함(Self-contained)하여 설계 결합도를 낮췄습니다.

# 의존성 패키지 설치
