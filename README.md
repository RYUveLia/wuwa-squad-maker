# 프로젝트: 명조(Wuthering Waves) 파티 시뮬레이터

## 1. 프로젝트 개요
*   **목적**: '명조: 워더링 웨이브'의 콘텐츠(특히 종말 매트릭스) 공략을 위해 사용자가 직접 파티 조합을 짜고 시각화할 수 있는 웹 툴 개발.
*   **핵심 모티브**: **TierMaker(티어메이커)**의 '드래그 앤 드롭' 방식과 '아이템 목록-등급 슬롯' UI 시스템을 벤치마킹함.

## 2. 주요 기능 및 UI 요구사항
*   **파티 배치 영역 (상단)**: 
    *   총 3개의 슬롯 (메인 딜러, 서브 딜러, 서포터).
    *   캐릭터 이미지를 드롭하여 배치. 이미 캐릭터가 있을 경우 교체(Replace) 로직 적용.
*   **캐릭터 풀 (하단)**: 
    *   보유한 모든 캐릭터 이미지가 나열되는 영역.
    *   필터 기능 필요 (속성: 회절, 응결 등 / 무기: 장검, 직검 등).
*   **카드 디자인**: 
    *   텍스트를 최소화하고 **캐릭터 이미지(정사각형 아이콘)** 중심의 심플한 디자인.
*   **저장 및 공유**:
    *   완성된 파티 조합을 이미지 파일로 저장하는 기능.

## 3. 데이터 로직 (Data)
*   **데이터 소스**: [명조 공식 위키(Fandom Wiki)](https://wutheringwaves.fandom.com/wiki/Character) 등 단일 사이트에서 캐릭터 아이콘 및 정보 추출.
*   **데이터 구조 (예시)**:
    ```json
    {
      "id": "jinshi",
      "name": "금희",
      "img": "/assets/characters/jinshi.png",
      "element": "Spectro",
      "weapon": "Broadblade"
    }
    ```

## 4. 기술 스택 (기술적 제안)
*   **Framework**: React (Vite 기반)
*   **Drag & Drop**: `@dnd-kit/core` (확장성 및 성능 우수)
*   **Styling**: Tailwind CSS (빠른 UI 구현)
*   **Image Export**: `html2canvas` (웹 영역 이미지 캡처)

## 5. Git 관리 가이드 (개인 프로젝트 분리)
*   **상황**: 회사 계정과 개인 계정을 한 기기에서 혼용 중.
*   **설정 방법**: 
    1.  프로젝트 로컬 폴더 내에서 `git config user.name/email` 설정 (Global 제외).
    2.  `.ssh/config` 설정을 통한 SSH 키 분리(IdentityFile 지정).
    3.  또는 `includeIf`를 사용하여 특정 폴더(`~/personal/`) 내 모든 프로젝트 자동 전환.
