# WuWa Matrix Squad Maker (종말 매트릭스 파티 빌더)

명조: 워더링 웨이브의 핵심 도전 콘텐츠인 **종말 매트릭스(Adversity Matrix)** 다중 파티 구성을 시각적으로 시뮬레이션하고, 다른 유저들과 간편하게 공유할 수 있는 웹 기반 파티 빌딩 도구입니다.

---

## ✨ 주요 기능 및 사용 방법 (Features & Usage)

### 1. 쉽고 직관적인 파티 배치
* **PC 환경 (드래그 앤 드롭)**
  * 좌측 도감에서 원하는 공명자 카드를 마우스로 끌어다 우측 파티 슬롯(딜러, 서브, 힐러)에 간편하게 배치할 수 있습니다.
  * 슬롯에 배치된 캐릭터를 직접 끌어서 다른 슬롯으로 옮겨 **위치를 서로 교환(Swap)할** 수 있습니다.
  * 배치된 캐릭터를 클릭하거나, 슬롯 바깥 영역으로 드래그하면 배치가 **즉시 해제**됩니다.
  * 드래그 핸들러를 사용해 **파티 행(Row) 단위로 자유롭게 순서를 드래그 정렬**할 수 있습니다.
* **모바일 환경 (터치 전용 바텀 시트)**
  * 모바일(1024px 미만 화면)에서는 무분별한 스크롤의 피로를 유발하던 드래그앤드롭 대신, **슬롯 터치 기반의 전용 캐릭터 선택 바텀 시트**가 자동으로 실행됩니다.
  * 한 명을 배치할 때마다 창이 닫히지 않고, 3명이 모두 꽉 찰 때까지 팝업을 유지해 주는 **연속 탭 편성**을 지원합니다.
  * 바텀 시트 상단의 **미니 3슬롯 실시간 프리뷰** 초상화를 터치해 실시간으로 배치를 취소할 수 있으며, 원하는 명수만큼만 구성하고 창을 닫을 수 있는 **'편성 완료' 확인 버튼**을 지원합니다.

### 2. 종말 매트릭스 출전 제한 규칙 자동화
* **서포터 최대 2회 중복 출전 허용:** 설지, 복링, 벨리나, 파수인, 모니에 등의 서포터 공명자는 최대 2개 파티까지 중복 편성이 가능합니다. 한도를 초과하여 배치할 경우 선입선출(FIFO) 규칙에 따라 가장 먼저 배치되었던 슬롯을 비우고 스마트하게 재배치합니다. (유출 정보 토글 설정에 따라 유출 정보 대상 공명자의 중복 편성 규칙도 유기적으로 자동 전환됩니다.)
* **시즌 특별 규칙 적용 (치사):** 3.5 시즌 임시 버프 조건인 **"치사 최대 2회 중복 출전"** 룰을 기본 탑재하여, 별도 조작 없이 치사도 최대 2회 배치가 가능하도록 자동 제어합니다.
* **방랑자 4속성 배타성 잠금 (Exclusion Lock):** 회절, 인멸, 기류, 전도 방랑자 중 하나라도 파티에 배치되면 나머지 3개 속성의 방랑자 카드는 도감에서 자동으로 비활성화(딤드) 처리되어 중복 출전이 원천 차단됩니다.

### 3. 보유 공명자 필터링 시스템 (Owned Filter)
* 내가 실제로 육성했거나 보유 중인 공명자들만 체크(✓)하여 관리할 수 있는 **보유 공명자 설정 모달**을 지원합니다.
* 설정한 보유 공명자 데이터는 브라우저 `localStorage`에 영구 기록되어 새로고침 후에도 그대로 유지됩니다.
* "보유한 공명자만 보기" 필터 스위치를 켜면, 리스트에서 미보유 캐릭터들이 제외되어 한층 기동력 있는 파티 빌딩이 가능합니다.

### 4. 유출 정보 ON/OFF 슬라이딩 토글 스위치 (Leak Filter Toggle)
* 미출시 유출 공명자의 노출 상태 및 제한 규칙을 손쉽게 제어할 수 있는 슬라이딩 토글 스위치를 지원합니다.
* 토글 ON 시 유출 설정 대상 공명자는 최대 2회 중복 출전이 허용되며, OFF 시에는 1회 출전으로 제한이 자동 연동됩니다.
* 설정한 토글 데이터 또한 브라우저 `localStorage`에 보존되어 새로고침 후에도 안정적으로 유지됩니다.

### 5. 만석 시 자동 파티 증설 시스템
* 모든 파티의 빈 슬롯이 가득 찬 상태에서 도감의 공명자 카드를 클릭했을 때, "편성할 빈 슬롯이 없습니다. 새로운 파티를 추가하고 배치하시겠습니까?" 라는 유려한 컨펌 모달이 출력됩니다.
* 확인 버튼을 누르면 자동으로 파티(Row)가 한 줄 신규 생성(증설)되면서 해당 캐릭터가 1번 슬롯(딜러)에 즉시 배치되어 끊김 없는 조작을 지원합니다.

### 6. 실수 방지용 확인 팝업 및 키보드 단축키 매핑 (ConfirmModal & Shortcuts)
* 파티원 해제, 개별 파티 줄 삭제, 툴바 전체 초기화 버튼, 그리고 만석 시 파티 추가 등 소중하게 짜놓은 스쿼드 배치가 우발적인 터치로 해제되는 비극을 예방하기 위해 아름다운 글라스모피즘 스타일의 **확인 컨펌 팝업창**을 제공합니다.
* 사용성 극대화를 위해 팝업창이 활성화된 상태에서 **`Enter` 키**를 누르면 즉시 승인(Confirm) 처리되고, **`Escape` 키**를 누르면 취소(Cancel) 처리되는 키보드 단축키 매핑을 완벽 지원합니다.

### 7. 조합 저장 및 코드 내보내기/불러오기
* **고화질 파티 캡처 이미지 저장:** 내가 구성한 조합 배치도를 고해상도 PNG 이미지 파일로 즉시 내려받을 수 있습니다.
* **편성 코드 공유 (Base64):** 영문 캐릭터 ID 기반의 Base64 인코딩 스트링 코드 공유를 지원하여, 간편하게 파티를 텍스트로 전파하고 역으로 복원해 올 수 있습니다. (기존 한글/영문 텍스트 코드 불러오기도 완벽 하위호환 지원)

## 💡 자주 묻는 질문 (FAQ) & 팁

* **Q. 열심히 짠 파티가 새로고침해도 유지되나요?**
  * A. **네, 안전하게 유지됩니다!** 구성하신 파티 편성 정보는 브라우저 `localStorage`에 실시간으로 자동 저장되므로, 새로고침을 하거나 웹 페이지를 완전히 닫았다가 다시 열어도 그대로 유지됩니다. 만약 조합을 다른 기기/브라우저로 옮기거나 영구 백업하고 싶다면 상단 툴바의 **"내보내기"** (텍스트 공유 코드 복사) 또는 **"이미지로 저장"** 기능을 활용해 주세요.
* **Q. 모바일 기기인데 드래그 앤 드롭이 작동하지 않습니다.**
  * A. 모바일 및 태블릿 기기(화면 너비 1024px 미만 해상도)에서는 사용 편의성을 위해 드래그 조작 대신 **원터치 바텀 시트** 팝업을 통해 파티원을 선택하도록 강제 설계되었습니다. 슬롯을 가볍게 한 번 터치하면 팝업이 활성화되니 터치로 간편하게 배치해 보세요!
* **Q. 첫 접속 시 봇 차단 보안 인증(Turnstile) 창이 계속 뜹니다.**
  * A. 무료 호스팅 환경의 악성 스크랩 봇이나 디바이스 공격 트래픽을 원천 차단하기 위한 비가시형 보안 캡차 솔루션입니다. 광고 차단 확장 프로그램(AdBlock 등) 사용으로 인해 캡차 API가 작동하지 않는 억울한 진짜 사용자를 위해, 하단에 비상용 **[보안 인증 건너뛰기]** 링크를 탑재했으니 이를 클릭해 우회 접속해 주세요.
