# WuWa Matrix Squad Maker (종말 매트릭스 파티 빌더)

명조: 워더링 웨이브의 핵심 도전 콘텐츠인 **종말 매트릭스(Adversity Matrix)** 다중 파티 구성을 시각적으로 확인하고, 다른 사람들과 간편하게 공유할 수 있는 웹 파티 빌딩 도구입니다.

---

## ✨ 주요 기능 및 사용 방법

### 1. 쉽고 직관적인 파티 배치
* **PC 환경 (마우스 끌어다 놓기)**
  * 왼쪽 도감에서 원하는 캐릭터 카드를 마우스로 끌어서 오른쪽 파티 슬롯(딜러, 서브, 힐러)에 간편하게 놓아 배치할 수 있습니다.
  * 배치된 캐릭터를 직접 마우스로 끌어서 다른 자리로 옮겨 **위치를 서로 교환할** 수 있습니다.
  * 배치된 캐릭터를 마우스로 클릭하거나 슬롯 바깥 영역으로 끌어다 놓으면 배치가 **즉시 취소**됩니다.
  * 왼쪽의 핸들을 마우스로 끌어 **파티 줄 단위로 자유롭게 순서를 변경**할 수 있습니다.
* **모바일 환경 (터치 전용 캐릭터 선택 창)**
  * 모바일 및 태블릿 등 화면이 작은 기기에서는 화면을 직접 끄는 대신, **자리를 터치하면 나타나는 전용 캐릭터 선택 창**을 통해 편성을 진행합니다.
  * 한 명을 선택할 때마다 창이 매번 닫히지 않고, 3명이 모두 채워질 때까지 열려 있는 **연속 선택 기능**을 지원합니다.
  * 선택 창 상단에 보이는 **실시간 미리보기** 초상화를 터치해 실시간으로 배치를 취소할 수 있으며, 원하는 인원만 구성하고 창을 닫을 수 있는 **'편성 완료' 버튼**을 제공합니다.

### 2. 출전 제한 규칙 자동 적용
* **서포터 최대 2회 중복 출전 허용:** 설지, 복링, 벨리나, 파수인, 모니에 등의 서포터 캐릭터는 최대 2개 파티까지 동시에 편성할 수 있습니다. 2회 한도를 초과하여 다른 파티에 추가 배치할 경우, 가장 먼저 배치했던 파티의 자리에서 자동으로 해제되어 편리하게 재배치됩니다. (유출 정보 설정 여부에 따라 미출시 캐릭터의 규칙도 자동으로 연동됩니다.)
* **시즌 특별 규칙 적용 (치사):** 3.5 시즌 임시 버프 조건인 **"치사 최대 2회 중복 출전"** 규칙이 기본으로 적용되어 있어, 별도 설정 없이 최대 2회 배치가 가능합니다.
* **방랑자 속성 상호 잠금:** 회절, 인멸, 기류, 전도 방랑자 중 하나라도 파티에 배치되면 나머지 3개 속성의 방랑자 카드는 도감에서 자동으로 어둡게 비활성화되어 중복 배치가 방지됩니다.

### 3. 보유 공명자 필터링 시스템
* 내가 실제로 육성했거나 가지고 있는 캐릭터들만 체크(✓)하여 관리할 수 있는 **보유 설정 창**을 지원합니다.
* 설정한 보유 캐릭터 정보는 사용 중인 브라우저에 저장되어 새로고침을 하거나 다시 방문해도 그대로 유지됩니다.
* "보유한 공명자만 보기" 스위치를 켜면, 내가 가지지 않은 캐릭터는 리스트에서 숨겨져 더욱 편리하게 파티를 구성할 수 있습니다.

### 4. 유출 정보 켜고 끄기 토글 스위치
* 미출시 공명자의 노출 여부와 규칙을 손쉽게 제어할 수 있는 토글 스위치를 지원합니다.
* 설정을 켜면 미출시 캐릭터가 도감에 표시되며 최대 2회 중복 출전이 허용되고, 끄면 1회 출전으로 제한이 자동 연동됩니다.
* 이 설정값 또한 사용 중인 브라우저에 저장되어 새로고침 후에도 유지됩니다.

### 5. 만석 시 자동 파티 추가 기능
* 모든 파티에 빈자리가 없을 때 도감의 캐릭터를 누르면, "새로운 파티를 추가하고 바로 배치하시겠습니까?" 라는 안내 창이 나타납니다.
* 승인 시 자동으로 파티 줄이 한 줄 더 생기면서 해당 캐릭터가 첫 번째 자리에 즉시 배치됩니다.

### 6. 조합 저장 및 코드 내보내기/불러오기
* **고화질 파티 이미지 저장:** 구성한 파티 배치도를 이미지(PNG) 파일로 다운로드하여 즉시 저장할 수 있습니다.
* **텍스트 코드 공유:** 파티 구성을 짧은 글자 코드로 변환하여 복사할 수 있습니다. 이 코드를 복사해 다른 사람에게 공유하면, '불러오기' 기능을 통해 동일한 파티를 즉시 다시 불러올 수 있습니다. (이전 형식의 공유 코드도 정상적으로 읽어옵니다.)

## 💡 자주 묻는 질문 (FAQ) & 팁

* **Q. 열심히 짠 파티가 새로고침해도 유지되나요?**
  * A. **네, 안전하게 유지됩니다!** 구성하신 파티 편성 정보는 사용 중인 인터넷 브라우저에 실시간으로 자동 저장되므로, 새로고침을 하거나 웹 페이지를 완전히 닫았다가 다시 열어도 그대로 유지됩니다. 만약 조합을 다른 기기나 다른 브라우저로 옮기고 싶다면 상단 메뉴의 **"내보내기"** (공유 코드 복사) 또는 **"이미지로 저장"** 기능을 활용해 주세요.
* **Q. 모바일 기기인데 마우스 끌어다 놓기가 작동하지 않습니다.**
  * A. 모바일 및 태블릿 기기(화면이 작은 환경)에서는 조작 편의성을 위해 직접 끄는 동작 대신 **선택 창**을 통해 파티원을 배치하도록 설계되어 있습니다. 빈자리를 가볍게 터치하여 캐릭터를 선택해 보세요!
* **Q. 첫 접속 시 보안 확인 화면이 계속 뜹니다.**
  * A. 악성 매크로나 비정상적인 사이트 공격을 막기 위한 보안 단계입니다. 일부 광고 차단 프로그램 등으로 인해 보안 확인이 진행되지 않는 경우, 화면 하단의 **[보안 인증 건너뛰기]** 링크를 누르면 즉시 사이트에 접속하실 수 있습니다.
