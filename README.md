# WuWa Matrix Squad Maker

명조: 워더링 웨이브의 핵심 도전 콘텐츠인 **종말 매트릭스(Tower of Adversity / Adversity Matrix)** 다중 파티 구성을 시각적으로 시뮬레이션하고 간편하게 공유할 수 있는 웹 기반 파티 빌더 도구입니다.

---

## ✨ 주요 기능 (Key Features)

1. **직관적인 드래그 앤 드롭 빌더 (Drag & Drop Builder)**
   * 좌측 도감에서 원하는 공명자 카드를 끌어다 우측 파티 슬롯(메인 딜러, 서브 딜러, 서포터)에 간편하게 배치할 수 있습니다.
   * 슬롯 간에 직접 끌어다 놓아 파티원의 **위치를 서로 교환(Swap)**할 수 있습니다.
   * 배치된 캐릭터를 클릭하거나, 슬롯 밖(도감 영역 등)으로 드래그 앤 드롭하면 배치가 **즉시 해제**됩니다.

2. **종말 매트릭스 출전 제한 규칙 완벽 자동화**
   * **서포터 최대 2회 중복 출전 허용:** 복링, 설지, 벨리나, 파수인, 모니에, 수수 등 서포터 공명자는 최대 2개 파티까지 중복 편성이 가능하며, 한도를 초과하면 FIFO(선입선출) 방식으로 기존 배치를 비우고 똑똑하게 재배치합니다.
   * **시즌 특별 규칙 적용 (치사):** 3.5 시즌 임시 버프 조건인 **"치사 최대 2회 중복 출전"** 룰을 기본 탑재하여, 별도 조작 없이 치사도 최대 2회 배치가 가능하도록 자동 제어합니다.
   * **방랑자 4속성 배타성 잠금 (Exclusion Lock):** 회절, 인멸, 기류, 전도 방랑자 중 하나라도 파티에 배치되면 나머지 3개 속성의 방랑자 카드는 도감에서 자동으로 딤드(비활성화) 잠금 처리되어 중복 출전이 방지됩니다.

3. **고화질 파티 캡처 이미지 저장**
   * `html2canvas` 연동을 통해 자신이 짠 조합 배치도를 **2배 선명도**의 고해상도 PNG 이미지로 저장할 수 있습니다. 
   * 커뮤니티나 디스코드에 간편하게 인증 샷을 남길 수 있습니다.

4. **파티 공유용 텍스트 코드 내보내기/불러오기 (Export/Import)**
   * 자신이 구성한 파티 리스트를 경량화된 JSON 텍스트 코드로 복사하여 메모해 두거나, 커뮤니티에서 다른 유저들의 파티 코드를 즉시 붙여넣어 시뮬레이터에 로드할 수 있습니다.

## 🛠️ 기술 스택 (Tech Stack)

* **프레임워크:** React 18, Vite 8, TypeScript 4
* **스타일링:** Tailwind CSS (Colocation 스타일 격리 패턴)
* **드래그 앤 드롭:** `@dnd-kit/core`
* **스크린샷 캡처:** `html2canvas`

---

## 🏛️ 아키텍처 및 코드 품질 (Architecture)

관심사의 완벽한 분리(SoC)와 컴포넌트 독립성을 최우선으로 리팩토링된 견고한 구조를 갖추고 있습니다:
* **UI View Layer ([src/App.tsx](src/App.tsx)):** 비즈니스 코드가 단 한 줄도 없이 UI 마크업 렌더링에만 집중하는 정갈한 뷰 파일입니다.
* **Logic Controller ([src/hooks/useSquadState.ts](src/hooks/useSquadState.ts)):** 스쿼드 상태값, 드래그앤드롭 이벤트 리스너, 중복 FIFO 계산, 복사/캡처 등 모든 비즈니스 규칙이 캡슐화된 커스텀 훅입니다.
* **Component-Style Colocation:** Emotion styled 컴포넌트 디자인을 벤치마킹하여, 전역 스타일 시트 없이 **각 컴포넌트 파일 하단부에 Tailwind CSS 전용 스타일 상수를 자가 포함(Self-contained)**하여 독립성을 확보했습니다.
